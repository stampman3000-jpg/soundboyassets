# Roadmap

## Project Status

Soundboy is an active research and development project.

Rather than following a fixed specification, the project evolves through
experimentation, testing and iteration. New ideas are explored regularly,
with successful concepts becoming part of the platform.

---

## Current Focus

Current development is centred around strengthening the core experience
before adding additional features.

Areas of focus include:

- Improving overall responsiveness
- Refining the user interface
- Expanding sequencing capabilities
- Continuing DSP experimentation
- Improving hardware integration

---

## In Progress

The following areas are currently being explored:

- [ ] Hardware assembly and enclosure
- [ ] Display optimisation
- [ ] Additional synthesis improvements
- [ ] Sample workflow refinement
- [ ] Performance optimisation

### User Experience

## Long-Term Exploration

Some ideas may become part of Soundboy in the future:

- USB audio
- Network connectivity
- Companion desktop tools
- Wireless synchronisation
- Alternative hardware platforms

These remain research topics rather than committed features.

---

## Design Philosophy

Every new feature should improve the experience of making music.

The project aims to remain focused, immediate and enjoyable to use rather
than accumulating features for their own sake.

Where possible, complexity should remain behind the scenes, allowing the
instrument itself to feel simple and approachable.
