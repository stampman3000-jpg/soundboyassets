# Hardware

## Overview

Soundboy is built around the idea of creating a dedicated musical instrument
rather than a general-purpose computer running music software.

The hardware choices are guided by a balance between accessibility,
flexibility and physical interaction. The aim is to create something that
feels immediate and hands-on while still allowing rapid experimentation.

---

## Core Platform

### Raspberry Pi

The Raspberry Pi provides enough processing power to experiment with:

- Custom audio engines
- Real-time sequencing
- Hardware interfaces
- Visual feedback
- MIDI communication

Using a single-board computer also keeps the project flexible, allowing
software ideas to be tested quickly before committing to more specialised
hardware.

---

## Interface Design

A key focus of Soundboy is the relationship between the musician and the
instrument.

Rather than relying on a touchscreen or mouse-driven interface, the project
explores physical controls:

- Dedicated buttons
- Rotary encoder input
- OLED display feedback
- Direct manipulation of musical parameters

The goal is to reduce friction between an idea and the sound being created.

---

## Prototype Hardware

Current hardware includes:

- Raspberry Pi 4
- 256x64 OLED display
- Custom button matrix
- Rotary encoder
- USB MIDI connectivity
- Prototype enclosure

## Iteration Process

The hardware has evolved through repeated testing and modification.

Early prototypes focused on proving individual ideas:
- Can the display provide enough feedback?
- Does the control layout feel natural?
- Is the interaction fast enough for performance?

Later iterations have focused on improving the relationship between the
physical controls, software architecture and musical workflow.

---

## Design Challenges

### Balancing flexibility and immediacy

A computer can offer almost unlimited options, but too many possibilities
can make an instrument feel less focused.

Soundboy explores the idea that constraints can improve creativity by
encouraging faster decisions and more direct interaction.
